# Cumplimiento Legal Colegios

Sitio estatico standalone para publicar en GitHub Pages.

## Contenido

- `index.html`: demo principal
- `nubox-logo.png`: recurso local usado por la demo
- `.nojekyll`: evita procesamiento innecesario de GitHub Pages

## Publicacion recomendada

Publicar este directorio como un repositorio propio y activar GitHub Pages desde la rama `main`, carpeta `/ (root)`.
